Verwaltung Impftermine

Hier können sie einen neuen Impftermin erstellen. Füllen Sie alle Felder aus und klicken Sie auf "Speichern". Möchten Sie den Vorgang abbrechen, klicken Sie auf "Abbrechen".

Wenn Sie einen Termin erstellen, den es schon in der Liste gibt, wird er zur entsprechenden Liste hinzugefügt.

Gibt es diesen Impftermin noch nicht, wird automatisch ein neuer Impftag erstellt. In diesem Fall ist die Angabe einer maximal möglichen Anmeldeanzahl erforderlich. Diese erfolgt im Feld "Mögliche Anmeldungen".
